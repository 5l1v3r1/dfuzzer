void df_init_introspection ( GDBusProxy *  dproxy,
char *  interface 
)

Gets introspection of object pointed by dproxy (in XML format), then parses XML data and fills GDBusNodeInfo representing the data. At the end looks up information about an interface and initializes module global pointers on first method and its first argument. This function must be called before using any functions from this module.

dproxyPointer on D-Bus interface proxy
interfaceName of application interface

void df_unref_introspection ( )

Call when done with this module functions (only after df_init_introspection() function call). It frees memory used by introspection_data (GDBusNodeInfo *) which is used to look up information about the interface (methods, their arguments, etc.).
